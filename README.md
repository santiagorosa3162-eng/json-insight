# JSON Insight

An interactive JSON validation and visualization tool built with vanilla JavaScript.

## Overview

JSON Insight allows users to validate, format and explore structured JSON data through an interactive tree interface. The tool focuses on clean architecture, recursive rendering and defensive parsing logic.

## Features

- JSON syntax validation  
- Beautify / formatting  
- Recursive tree view rendering  
- Expand / collapse nodes  
- Structural statistics (keys, arrays, depth, primitives)  
- Graceful error handling  
- Zero external dependencies  

## Technical Focus

- Recursive data traversal  
- Safe JSON parsing  
- Dynamic DOM rendering  
- Efficient structural analysis  
- Lightweight single-file architecture  

## Tech Stack

- HTML  
- CSS  
- Vanilla JavaScript
